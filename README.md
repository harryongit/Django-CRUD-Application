# Django CRUD Application

This is a simple Django application that demonstrates Create, Read, Update, and Delete (CRUD) operations.

## Features

- Create new records
- Read and display records
- Update existing records
- Delete records

## Technologies Used

- Python
- Django
- SQLite (default database)
- HTML/CSS (for frontend)
- Bootstrap (for styling)

## Prerequisites

- Python 3.x
- Django 3.x or higher

## Installation

1. Clone the repository:

   ```sh
   git clone https://github.com/harryongit/Django-CRUD-Application.git
   cd Django-CRUD-Application
   ```

2. Create and activate a virtual environment:

   ```sh
   python -m venv venv
   source venv/bin/activate   # On Windows use `venv\Scripts\activate`
   ```

3. Install the required dependencies:

   ```sh
   pip install -r requirements.txt
   ```

4. Apply migrations to set up the database:

   ```sh
   python manage.py migrate
   ```

5. Create a superuser to access the Django admin:

   ```sh
   python manage.py createsuperuser
   ```

6. Run the development server:

   ```sh
   python manage.py runserver
   ```

7. Open your web browser and go to `http://127.0.0.1:8000` to view the application.

## Usage

- Access the Django admin at `http://127.0.0.1:8000/admin` to manage records.
- Use the application interface to create, read, update, and delete records.
